Package for implementations of Pix Captchas : Captcha with pics chalenges.
It defines three abstract classes and a concrete one:
  • PixCaptcha : ok we got it!
  • PixCaptchaFactory : Base class for PixCaptcha generation
  • PixCaptchaEngine : Base class for PixCaptchaFactory management
  • DefaultPixCaptchaEngine : very simple implementation of the class below, using an array of PixCaptchaFactory to be constructed and randomly returns one.
