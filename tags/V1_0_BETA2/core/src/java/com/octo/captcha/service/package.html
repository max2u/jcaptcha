This is the base package for JCAPTCHA service implementation.
It defines two main interfaces :
  • AbstractCaptchaService : The base implementation of a service, using two interfaces defined below, and a CaptchaEngine.
  • CaptchaStore : The stock of captcha used by the service to store captchas from the time it is generated to the time the captcha validation routine is called
  • IDGenerator : A Unique ID generator used by the service to mark captchas
